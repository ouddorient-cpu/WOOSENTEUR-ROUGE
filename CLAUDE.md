## 3. Prompts Maîtrisés (Optimisés via Promptomatix)

Utilise ce framework pour TOUS les outputs :
- **[TASK]** : Objectif précis.
- **[INSTRUCTIONS]** : Étapes Chain-of-Thought.
- **[RULES]** : SEO/Conversion/Humain.
- **[OUTPUT_FORMAT]** : Structuré (JSON/MD).

Priorise spécificité, cadrage positif, exemples few-shot en fin.

### Posts Sociaux Viraux

```
[TASK] Crée 5 posts viraux pour [plateforme] sur [niche/produit], objectif 10%+ engagement.

[INSTRUCTIONS]
1. Hook 3s (question/choc/stats).
2. Storytelling AIDA (Attention-Intérêt-Désir-Action).
3. Hashtags/CTA forts.
4. Adapte ton [TikTok: court fun | LinkedIn: pro | Insta: visuel | FB: communauté].

[RULES]
- Ton Marc Lefèvre : humain expert.
- SEO : mots-clés naturels (parfums Lattafa, SaaS e-commerce).
- Conversion : lien bio/stripe.
- Longueur : [TikTok 15s script | autres 100-200 mots].

[OUTPUT_FORMAT]
- Titre | Script/Texte | Visuel idée | Hashtags | CTA
```
