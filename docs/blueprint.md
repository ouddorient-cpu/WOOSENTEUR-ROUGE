# **App Name**: WooSenteur AI

## Core Features:

- AI Product Description Generation: Generates SEO-optimized product descriptions for beauty products using the Gemini model via Genkit, following strict reasoning logic and guardrails. It utilizes a tool that validates product existence and extracts attributes before generating content.
- Product Image Validation: Validates the uploaded product image to ensure it matches the product description, enhancing accuracy and preventing mismatches. Leverages AI to analyze the image and compare it to the generated description.
- WooCommerce Export: Enables direct export of product data to WooCommerce via the WooCommerce REST API, streamlining the product listing process.
- CSV Export: Allows users to export selected product data into a CSV file for flexible data management and use in other platforms.
- Subscription Management via Stripe: Integrates Stripe Checkout and Customer Portal for managing user subscriptions and handling payments, offering different tiers with varying credits and features.
- Product Dashboard: Displays a dashboard listing all generated products with options to edit, export, and manage product information.
- User Authentication: Implements Firebase Authentication with email/password and magic link options for secure and easy user login.

## Style Guidelines:

- Primary color: Amaranth red (#C1292E) for a luxurious feel, drawing inspiration from high-end beauty brands. This deep red provides a sense of sophistication.
- Background color: Blush pink (#F8E7EB), a very desaturated tint of the primary color, creating a soft and elegant backdrop reminiscent of beauty product packaging.
- Accent color: Apricot (#F46036), a vibrant analogous hue to the primary color, to draw attention to key UI elements such as call-to-action buttons.
- Headline font: 'Playfair', a serif font for titles, lending a touch of elegance and classic appeal.
- Body font: 'PT Sans', a sans-serif font to ensure readability and maintain a clean, professional look.
- Clear, luxurious, and professional layout to enhance user experience, mirroring the sophistication of the beauty industry.
- Use minimalistic icons from Lucide-react to ensure clarity and consistency.
